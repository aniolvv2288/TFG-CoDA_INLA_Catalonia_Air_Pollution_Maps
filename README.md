# Final Degree Project (TFG) – Anàlisi de dades composicionals (CoDA) per a la cartografia de la contaminació atmosfèrica a Catalunya

This repository contains the code, data workflows, and documentation for my Final Degree Project (TFG) in Applied Statistics at Universitat Autònoma de Barcelona (UAB).

## Project Overview

The main objective of this project is to apply Compositional Data Analysis (CoDA) techniques to characterize and map air pollution across Catalunya. The goal is to generate continuous environmental exposure maps that not only describe the spatial distribution of pollutants, but also serve as a foundation for future epidemiological studies—such as those investigating associations between air pollution exposure and mental health or cardiovascular diagnoses.
